## How to Run

1. Clone the repository:
   ```sh
   git clone https://github.com/singhsatyaprakash/DSA-With-CPP.git
   ```
2. Navigate to the project directory and compile the desired `.cpp` file:
   ```sh
   cd DSA-With-CPP
   g++ src/main.cpp -o main
   ./main
   ```
